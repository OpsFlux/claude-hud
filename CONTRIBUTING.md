# 贡献指南

感谢你为 Claude HUD 做出贡献。本仓库规模较小且迭代很快，因此我们更强调表述清晰与快速评审。

## 如何贡献

1) Fork 并 clone 仓库
2) 创建分支
3) 进行修改
4) 运行测试，并在需要时更新文档
5) 提交 Pull Request

## 开发

```bash
npm ci
npm run build
npm test
```

## 测试

完整的测试策略、fixtures 和快照更新方式请参见 `TESTING.md`。

## 代码风格

- 修改尽量聚焦且小步提交。
- 行为变更优先补充测试。
- 除非必要，避免引入新的依赖。

## Pull Request

- 描述问题与修复方案。
- 附带测试，或说明为何不需要测试。
- 相关时请关联对应 issue。
