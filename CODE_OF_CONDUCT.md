# 行为准则

## 我们的承诺

我们承诺让每个人在参与本社区时都能获得无骚扰的体验，不论其年龄、体型、可见或不可见的残障状况、族裔、性别特征、性别认同与表达、经验水平、教育程度、社会经济状况、国籍、外表、种族、宗教，或性取向与性别认同。

## 我们的标准

有助于营造积极环境的行为示例包括：

- 尊重他人、体贴周到
- 使用友善且包容的语言
- 接受建设性的反馈
- 关注对社区最有利的事情

不可接受的行为示例包括：

- 骚扰或歧视
- 钓鱼、侮辱或贬损性言论
- 未经允许发布他人的私人信息

## 执行

社区维护者负责澄清可接受行为的标准，并可针对不可接受的行为采取适当行动。

如遇辱骂、骚扰或其他不可接受的行为，可通过联系维护者进行举报：jarrodwttsyt@gmail.com。
